---
title: Ember._Input
until: 7.0.0
since: 6.5.0
parent: deprecate-import-ember-from-ember
---


Previously, `_Input` could be accessed via the `Ember` import:
```js
import Ember from 'ember';

Ember._Input
```

If needed, `_Input` can be imported:
```js
import { Input }  from '@ember/component';
```
