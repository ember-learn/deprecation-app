---
title: Ember.Evented
until: 7.0.0
since: 6.5.0
parent: deprecate-import-ember-from-ember
---


Previously, `Evented` could be accessed via the `Ember` import:
```js
import Ember from 'ember';

Ember.Evented
```

If needed, `Evented` can be imported:
```js
import Evented from '@ember/object/evented';
```
