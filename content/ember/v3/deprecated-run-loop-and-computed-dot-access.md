---
title: Run loop and computed dot access
until: 4.0.0
since: "3.27"
---


Using `.` to access computed or run loop functions has been deprecated, such
as `computed.filter`.
Instead, import the value directly from the module:

```js
import { filter } from '@ember/object/computed';
```

---
